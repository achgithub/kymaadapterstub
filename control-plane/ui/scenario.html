Quick Reference

Quick Start:

  1. Add adapters with desired behavior
  2. Click "Launch All Adapters"
  3. Copy URLs from the table
  4. Configure CPI with these URLs

Adapter Types:

  • REST: HTTP endpoint, configurable status code and response body
  • OData: OData v2/v4 endpoint, returns OData XML/JSON
  • SOAP: SOAP 1.1/1.2 endpoint, returns envelope or fault
  • XI: SAP PI/XI SOAP, returns XI routing ACK or fault
  • SFTP: SFTP server with configurable files and auth
  • AS2: B2B messaging, returns sync MDN acknowledgement
  • AS4: ebMS3 messaging, returns AS4 Receipt Signal
  • EDIFACT: EDI (auto-detects EDIFACT/X12), returns CONTRL or 997

Sender Adapters (Outbound):

  • REST-SENDER: POSTs to a target HTTP URL, returns response
  • SOAP-SENDER: POSTs SOAP envelope to target, sets SOAPAction header
  • XI-SENDER: POSTs SAP XI/PI SOAP with routing headers to target

Behavior Modes:

  • Success: Return configured success response
  • Failure: Return configured error response
