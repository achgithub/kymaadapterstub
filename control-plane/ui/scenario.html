Quick Reference

Quick Start:

  1. Add adapters with desired behavior
  2. Click "Launch All Adapters"
  3. Copy URLs from the table
  4. Configure CPI with these URLs

Adapter Types:

  • REST: HTTP/HTTPS endpoint with configurable status codes and responses
  • SFTP: SFTP server with files and authentication control
  • OData: OData service endpoint

Behavior Modes:

  • Success: Return configured success response
  • Failure: Return configured error response
